# CottonBall servers

## How to make a CottonBall server?

1. Open <a href="https://dashboard.render.com/login">Render</a> and log in.
2. Go to your project and create a web service (New service - Web service).
3. Choose "Public Git Repository" and paste this as the URL: "https://github.com/cbserverz/cbserverz.github.io"
4. Change name to "cottonmain"
5. Change the language to "Node"
6. Change the region to the region closest to you.
7. Change the build command to "npm install ws http".
8. Change the start command to "node cottonmain.min.js".
9. Change instance type to your plan.
10. Press "Deploy Web Service".
